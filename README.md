# sample-cypress
Sample code using cypress with cucumber

## Run this project
Open cypress
```sh
npm run open:cypress
```

Open cypress with specific viewport dimensions
```sh
npm run open:cypress:mobile
```

Run cypress in a CI like context
```sh
npm run run:cypress
```
